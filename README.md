# Permutations-of-the-characters-in-a-string-Java
Given a string the program prints all possible permutations of its characters.

Format-<br>
Input - The string<br>
Output - The permutations<br>

Example-<br>
Input:<br>
abc<br>
Output:<br>
[a, b, c]<br>
[a, c, b]<br>
[b, a, c]<br>
[b, c, a]<br>
[c, b, a]<br>
[c, a, b]<br>
